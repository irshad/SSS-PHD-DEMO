What is renarration?

By renarration we mean that current, already published web content should be re-presented, re-packaged and re-delivered in novel ways. The renarration should be controlled by 3rd party volunteers who are other than the author of the content, or the end-user of the content.

What is semantic style sheet?

This style sheet is a declarative language (perhaps even a DSL) that tells the processor what to do. We can have a frequent variation of the SSS. The purpose of this semantic style sheets is to transform web pages at the content, behavior and presentation levels. SSS is just like a CSS i.e., a page or more of script written in a text file format.
